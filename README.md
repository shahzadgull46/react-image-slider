# 🖼️ Image Slider

A modern and responsive **Image Slider application** built using React.  
This project demonstrates interactive UI development, React components, state management, and dynamic image rendering.

---

## 🚀 Features

- Responsive image slider layout
- Next and Previous image navigation
- Smooth image switching
- Dynamic image rendering
- Reusable React components
- Mobile-friendly design
- Clean and professional interface

---

- JavaScript (ES6+)
- Tailwind CSS / CSS

## 📖 How It Works

The application stores images in an array and uses React state to track the currently displayed image.  
Clicking **Next** moves to the next image, while clicking **Previous** returns to the previous image.  
The UI updates instantly without requiring a page refresh.
